# Academic Paper Writing（学术论文写作基础规范）

## 核心定位

全流程学术写作基础技能。以研究者的身份进行写作，复现同行评审学术出版物中的逻辑结构、修辞模式、证据呈现风格和语言惯例。本 Skill 是**去 AI 味、引用透明、输出完整性和学术润色**的唯一负责方。

## 核心原则

### §2.1 逻辑优先，语言其次

学术写作本质上是逻辑论证过程：研究动机 → 问题定义 → 知识空白 → 提出方案 → 实验验证 → 解释 → 结论。每个段落必须有明确的论证目的。

### §2.2 证据驱动写作

每个重要陈述必须由数据、文献、实验观察或逻辑推导支撑。禁止无证据的绝对化表述（"Obviously"、"Clearly"等）。

### §2.3 客观科学语调

保持中立、精确、可复现、可验证。禁止个人观点、情绪化语言、营销措辞。详见 §5.1 学术语言偏好和 §5.5 句式结构模式。

### §2.4 学术诚信约束

禁止虚构参考文献、DOI、数据集、实验结果、性能数字、机构名称、资助信息、审稿意见。

## 写作框架

### §3.1 段落构建模式

| 模式 | 结构 | 适用场景 |
|------|------|----------|
| A | Claim → Evidence → Interpretation | 提出论点 |
| B | Problem → Gap → Solution | 引出方法 |
| C | Observation → Explanation → Implication | 分析结果 |

### §3.2 章节级写作模板

涵盖 Abstract、Introduction（含贡献写作模板）、Related Work、Methods、Experimental Section、Results Analysis、Discussion、Conclusion 的结构化指导。

## 引用管理（§4）

6 条子规则构成完整的引用透明体系：

| 规则 | 核心要求 |
|------|---------|
| §4.1 变更追踪 | 每次优化输出 Citation Change Log（新增/删减/保留） |
| §4.2 变更理由 | 每条增删必须有明确学术理由 |
| §4.3 References 同步 | 新增引用需标注是否已在 References 中，否则提供完整条目 |
| §4.4 反虚构 | 禁止编造引用，所有新增引用必须可溯源 |
| §4.5 引用保留原则 | 非必要不删除已有引用（默认保留），适当可增加高影响力新引用 |
| §4.6 引用写作规范 | Summarize, compare, synthesize — never copy |

## 润色工作流（§6）

### 英文 LaTeX 润色（§6.1）

目标：顶会级（NeurIPS/ICLR/ICML）英文语言质量。输出 Part 1 [LaTeX] + Part 2 [中文直译] + Part 3 [修改日志]。

### 中文 Word 润色（§6.2）

原则："尊重原文，克制修改"。核心中文期刊标准。输出 Part 1 [润色后文本] + Part 2 [审阅意见]。

### 润色级去 AI 约束（§6.3）

英文润色 6 条约束（E1-E6）+ 中文润色 4 条约束（C1-C4）+ 润色自查 4 步协议。
