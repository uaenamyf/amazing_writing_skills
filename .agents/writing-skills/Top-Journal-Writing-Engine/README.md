# Top-Journal Writing Engine（顶级期刊写作引擎）

## 核心定位

生成在逻辑严谨性、科学叙事策略、审稿人期望和发表标准方面**媲美顶级期刊和会议**的稿件。

> **目标不是最大化语言复杂度，而是最大化可发表性。**

---

## 核心哲学

| 层次 | 论文解释的内容 |
|------|---------------|
| 低层次 | "我们做了什么" |
| 中层次 | "为什么它有效" |
| 高层次 | "为什么这个问题重要、为什么现有方案不足、为什么这个方案有本质区别、为什么研究发现推动领域前进" |

### 优先级排序

```
科学意义 > 新颖方法 > 实验性能
```

---

## 核心模块

### 1. 科学叙事引擎

写作前构建完整故事链：

```
问题重要性 → 科学挑战 → 知识空白 → 核心洞察 → 提出方案 → 实验证据 → 科学含义 → 未来影响
```

每个章节必须服务于这个故事，不允许孤立段落或脱节实验。

### 2. 新颖性放大框架

新颖性**不能声称**，必须**展示**。对每个贡献按以下结构解释：

```
现有方法 → 缺失什么 → 为什么该局限重要 → 方案如何解决 → 证据支持
```

### 3. 高影响力引言生成器

顶级引言必须创造紧迫感，结构分为 7 层：

1. 该领域为何重要
2. 该问题为何重要
3. 现有方法为何不足
4. 解决该问题为何困难
5. 核心科学洞察
6. 提出方案
7. 贡献

### 4. 贡献优化引擎（SMNV 标准）

每个贡献必须满足：

| 标准 | 英文 | 说明 |
|------|------|------|
| 具体 | Specific | 明确可操作的内容 |
| 可衡量 | Measurable | 可量化评估的成果 |
| 新颖 | Novel | 与现有工作的本质区别 |
| 必要 | Necessary | 对领域不可或缺的价值 |
| 可验证 | Verifiable | 可通过实验或逻辑验证 |

## 章节级写作优化

### 顶级摘要生成器

结构：背景 → 知识空白 → 方法 → 结果 → 科学意义。以 `The findings provide insights into...` 收尾，禁止用 `Future work will...` 结尾。

### 顶级方法呈现

方法章节应显得"必然"——读者应感到"这个设计自然地从问题推导而来"。对每个模块按：问题 → 设计动机 → 实现 → 预期效果 → 数学公式 呈现。

### 机制导向写作

顶级期刊偏好"机制解释"而非"架构描述"。解释为什么模块有效、为什么信息流高效、为什么表征变得更具区分性。

### 实验设计智能

每个贡献必须有对等证据（Contribution A → Ablation A, Contribution B → Ablation B）。禁止无支撑声明。

### 基准策略

优先 SOTA 方法、近期方法、强基线、社区基准。顶级审稿人重视公平性。

### 统计报告引擎

始终报告均值、标准差、置信区间、统计显著性。避免单次运行结果和挑选样本。

### 讨论卓越层

讨论章节往往是接收与拒稿的分水岭。包含：为什么会发生这些结果、为什么会出现失败、实际含义、理论含义、局限性、未来机会。

### 局限性写作框架

顶级期刊期望透明度。模板：`Although the proposed framework achieves promising performance, several limitations remain. First, ... Second, ... Third, ... Addressing these issues constitutes an important direction for future research.`

## 审稿人期望模拟

内部评估 6 个维度：新颖性、技术质量、实验质量、可复现性、影响力、清晰度。每个维度在定稿前需达到 ≥8/10。

## 期刊特定画像

| 期刊/会议 | 聚焦 |
|----------|------|

| IEEE TPAMI | 基础方法论、理论严谨性、强消融、通用框架 |
| Nature Communications | 科学发现、机理解释、广泛影响、跨学科意义 |
| CVPR/ICCV/ECCV | 基准主导、新颖架构、可视化证据、大规模实验 |
| NeurIPS/ICML/ICLR | 理论基础、优化、泛化、数学严谨性 |

## AI 写作抑制

系统性去 AI 味（词汇替换、连接词修剪、破折号控制、句式模式检测、8 条自查协议）统一由 [`Academic-Paper-Writing`](../Academic-Paper-Writing/) 负责。本 Skill 聚焦可发表性最大化，不处理 AI 语言模式移除。

---

## 科学影响力生成器

将发现连接到：科学理解 → 方法论进步 → 实际部署 → 跨领域适用性 → 长期研究机遇。顶级论文推动领域前进，不仅仅贡献数据集或基准。

---

## 发表准备度检查

输出前验证 5 条链：逻辑一致性链 → 新颖性链 → 证据链 → 贡献链 → 审稿人说服链。任何链条不完整则修改后再生。
