# Top-Journal Writing Engine（顶级期刊写作引擎）

## 核心定位

生成在逻辑严谨性、科学叙事策略、审稿人期望和发表标准方面**媲美顶级期刊和会议**的稿件。

> **目标不是最大化语言复杂度，而是最大化可发表性。**

---

## 章节结构（7 章）

| 章 | 标题 | 核心内容 |
|---|------|---------|
| §1 | Mission & Core Philosophy | 使命定义 + 三层论文哲学（做了什么 → 为什么有效 → 为什么重要） |
| §2 | Scientific Storytelling Framework | 故事链 + 新颖性放大框架 + 科学影响力生成器 |
| §3 | Section-Level Writing | 引言（7 层）、贡献（SMNV 标准）、摘要、方法、机制导向写作 |
| §4 | Experimental Excellence | 实验设计智能、基准策略、统计报告、讨论卓越层、局限性框架 |
| §5 | Journal-Specific Profiles | 7 大领域期刊/会议风格画像（综合科学/生物生态/生信/CS/工程/医学影像/遥感） |
| §6 | Quality Assurance | 审稿人期望模拟、AI 写作抑制、发表准备度检查、5 级输出等级 |
| §7 | Cross-Reference | 与其他 Skill 的协作关系 |

---

## 科学叙事框架（§2）

### §2.1 故事链

问题重要性 → 科学挑战 → 知识空白 → 核心洞察 → 提出方案 → 实验证据 → 科学含义 → 未来影响。每个章节必须服务于这个故事。

### §2.2 新颖性放大

新颖性**不能声称**，必须**展示**。对每个贡献按：现有方法 → 缺失什么 → 为什么该局限重要 → 方案如何解决 → 证据支持 的结构展开。

### §2.3 科学影响力生成器

将发现连接到：科学理解 → 方法论进步 → 实际部署 → 跨领域适用性 → 长期研究机遇。

## 章节级写作（§3）

| 子节 | 核心方法 |
|------|---------|
| §3.1 高影响力引言 | 7 层结构创造紧迫感 |
| §3.2 贡献优化 | SMNV 标准（Specific/Measurable/Novel/Necessary/Verifiable）|
| §3.3 顶级摘要 | 背景 → 知识空白 → 方法 → 结果 → 科学意义，禁止 Future work 结尾 |
| §3.4 顶级方法呈现 | 方法章节应显得"必然"：问题 → 设计动机 → 实现 → 预期效果 → 数学公式 |
| §3.5 机制导向写作 | 解释为什么有效，而非仅描述架构 |

---

## 实验卓越（§4）

| 子节 | 核心要求 |
|------|---------|
| §4.1 实验设计智能 | 每个贡献必须有对等证据（A → Ablation A），禁止无支撑声明 |
| §4.2 基准策略 | 优先 SOTA/近期/强基线/社区基准 |
| §4.3 统计报告 | 始终报告均值、标准差、置信区间、统计显著性 |
| §4.4 讨论卓越层 | 为什么发生 + 为什么失败 + 实际/理论含义 + 局限 + 未来机会 |
| §4.5 局限性框架 | 透明化模板：Although...several limitations remain. First...Second...Third... |

## 期刊特定画像（§5）

本 Skill 是 7 大领域期刊/会议风格画像的**唯一维护方**：

| 领域 | 代表期刊 | 聚焦关键词 |
|------|---------|-----------|
| 综合科学 | Nature, Science, PNAS, Nat. Commun., Sci. Adv. | 科学突破、跨学科影响、机制深度 |
| 生物与生态 | Curr. Biol., eLife, Ecol. Lett., Biol. Conserv., Methods Ecol. Evol., Behav. Ecol., J. Anim. Ecol. | 机制理解、进化意义、保护应用、方法创新 |
| 生物信息学 | Bioinformatics, PLOS Comput. Biol., Brief. Bioinform. | 算法创新、生物学验证、可复现性 |
| 计算机科学 | IEEE TPAMI, CVPR/ICCV/ECCV, NeurIPS/ICML/ICLR | 基础方法论、基准主导、理论严谨 |
| 工程与信号处理 | IEEE TSP, JASA, IEEE/ACM TASLP | 数学严谨、算法新颖、实验验证 |
| 医学影像 | Med. Image Anal., IEEE TMI | 临床相关、可解释性、多中心鲁棒 |
| 遥感与地理空间 | ISPRS J., Remote Sens. Environ. | 地理空间意义、跨区域迁移、多源融合 |
