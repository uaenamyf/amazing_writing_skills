# Author Style Fingerprint Engine（作者风格指纹引擎）

## 核心定位

学习并复现特定作者、研究组、实验室或出版物谱系的高层写作特征。

> **目标不是模仿措辞，而是建模作者的科学思维过程、论证策略、稿件组织和沟通风格。**

## 指纹提取层（§3）

### §3.1 科学思维画像

分析作者如何切入研究问题，分类为：理论驱动型 / 问题驱动型 / 应用驱动型 / 机制驱动型 / 数据驱动型 / 混合型。

### §3.2 引言指纹

分析引言的构建方式：背景深度、动机强度、空白构建风格、贡献位置、叙事策略。识别 4 种引言原型（Type A/B/C/D），**复用原型而非复用措辞**。

### §3.3 贡献指纹

分析贡献的呈现方式：贡献数量与粒度、新颖性框架、自信程度、证据风格。含 8 种贡献类型分类。

### §3.4 修辞策略提取

新颖性策略、鲁棒性策略、泛化策略、重要性策略、可解释性策略的提取与等效替代生成。

### §3.5 语言风格画像

构建量化风格画像：句子级指标（句长/方差）、词汇偏好（动词/过渡词/hedge 密度）、结构模式（段落/章节/图表/公式密度）。

## 风格迁移（§4）

### §4.1 风格迁移约束矩阵

| 维度 | 目标相似度 |
|------|-----------|
| 逻辑相似度 | 80–95% |
| 结构相似度 | 70–90% |
| 语言相似度 | ≤30% |
| 句子相似度 | ≤15% |
| 直接短语复用 | 0% |

### §4.2 反抄袭规则

禁止复制：句子、段落、标题、贡献陈述、摘要、段落过渡、公式解释。即使明确要求也不例外。

### §4.3 多作者融合模式

从多篇论文分别提取引言逻辑、方法逻辑、实验逻辑、讨论逻辑，融合为新的复合画像。

## 输出模式（§6）

| 模式 | 说明 |
|------|------|
| Mode 1 | 风格分析报告（仅指纹） |
| Mode 2 | 章节级仿写 |
| Mode 3 | 全文仿写 |
| Mode 4 | 期刊特定仿写（**默认**） |
| Mode 5 | 作者特定仿写 |

优先级：逻辑 > 结构 > 论证 > 语言
