# Author Style Fingerprint Engine（作者风格指纹引擎）

## 核心定位

学习并复现特定作者、研究组、实验室或出版物谱系的高层写作特征。

> **目标不是模仿措辞，而是建模作者的科学思维过程、论证策略、稿件组织和沟通风格。**

---

## 优先级排序

```
推理模式 > 科学叙事 > 结构偏好 > 修辞习惯 > 语言特征
```

高层思维模式优先于表层语言特征。

---

## 作者指纹构建流程（输入 3~20 篇代表性论文）

### 第一层：科学思维画像

分析作者如何切入研究问题，分类为：

| 类型 | 英文 | 特征 |
|------|------|------|
| 理论驱动型 | Theory-Driven | 从理论出发推导方法 |
| 问题驱动型 | Problem-Driven | 从实际问题出发寻找方案 |
| 应用驱动型 | Application-Driven | 以应用场景为核心 |
| 机制驱动型 | Mechanism-Driven | 以机理解释为主线 |
| 数据驱动型 | Data-Driven | 以数据观察为起点 |
| 混合型 | Hybrid | 多种策略融合 |

### 第二层：引言指纹

分析引言的构建方式，提取以下维度：

- **背景深度**（Background Depth）
- **动机强度**（Motivation Strength）
- **空白构建风格**（Gap Construction Style）
- **贡献位置**（Contribution Placement）
- **叙事策略**（Storytelling Strategy）

识别 4 种引言原型（Type A / B / C / D），**复用原型而非复用措辞**。

### 第三层：贡献指纹

分析贡献的呈现方式：

- 贡献数量与粒度
- 新颖性框架
- 自信程度
- 证据风格

---

## 关键约束

- ✅ 建模科学思维过程
- ✅ 复用论证策略与结构模式
- ❌ 禁止模仿措辞
- ❌ 禁止句子级复制

---

## 语言风格画像

构建量化风格画像：

- **句子级**：平均句长、短句/长句比率、句长方差
- **词汇偏好**：动词频率排名、过渡词分布、hedge 密度
- **结构模式**：段落长度分布、章节组织惯例

---

## 期刊指纹识别

| 期刊/会议 | 风格向量 |
|----------|---------|
| IEEE | 简洁、技术性、高密度、低叙事 |
| ACM | 算法导向、形式化、结构化 |
| Nature/Science | 高层意义、广泛影响、机理解释 |
| CVPR/ICCV/ECCV | 视觉证据、强消融、基准导向 |
| NeurIPS/ICML/ICLR | 理论动机、数学严谨、泛化讨论 |

---

## 风格迁移约束

| 维度 | 目标相似度 |
|------|-----------|
| 逻辑相似度 | 80–95% |
| 结构相似度 | 70–90% |
| 语言相似度 | ≤30% |
| 句子相似度 | ≤15% |
| 直接短语复用 | 0% |

确保风格迁移不产生抄袭。

---

## 与其他 Skill 的协作

- **去 AI 味 / 润色**：由 [`Academic-Paper-Writing`](../Academic-Paper-Writing/) 负责。本 Skill 专注作者风格建模，不处理文本去 AI 化。
- **顶级期刊优化**：将提取的指纹作为 [`Top-Journal-Writing-Engine`](../Top-Journal-Writing-Engine/) 的输入。
- **审稿模拟**：将指纹与 [`Reviewer-Simulation`](../Reviewer-Simulation%26Rejection-Prediction-Engine/) 的期望交叉验证。

## 输出模式

| 模式 | 说明 |
|------|------|
| Mode 1 | 风格分析报告 |
| Mode 2 | 章节级仿写 |
| Mode 3 | 全文仿写 |
| Mode 4 | 期刊特定仿写（默认） |
| Mode 5 | 作者特定仿写 |

优先级：逻辑 > 结构 > 论证 > 语言
