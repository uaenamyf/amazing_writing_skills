# Paper Reverse Engineering（论文逆向工程）

## 核心定位

当提供目标论文时，**不直接模仿措辞**，而是逆向工程其底层学术写作策略，并使用提取的风格模式重构新内容。

---

## 核心原则

| 原则 | 说明 |
|------|------|
| ✅ 保留推理模式 | 复用论文的逻辑推导方式 |
| ✅ 保留修辞结构 | 复用论证框架与段落组织 |
| ✅ 保留科学论证风格 | 复用证据呈现与说服策略 |
| ❌ 避免文本相似 | 不产生近义改写 |
| ❌ 避免抄袭 | 不复制原创表述 |
| ❌ 避免句子级模仿 | 不复用具体措辞 |

---

## 逆向工程流程

### 阶段一：结构分解

识别论文类型并构建**论文蓝图**：

- 论文类型：方法论文 / 应用论文 / 基准论文 / 综述论文 / 理论论文 等
- 研究问题 → 核心假设 → 科学动机 → 贡献策略 → 验证策略 → 预期审稿人关注点

### 阶段二：引言逻辑提取

逐层分析动机构建过程：

```
背景层 → 当前进展 → 现有局限 → 知识空白 → 研究问题 → 提出方案 → 贡献
```

识别引言模式库中的具体模式（Pattern A / B / C），**复用模式而不复用措辞**。

| 模式 | 结构 |
|------|------|
| Pattern A | 广义问题 → 狭义问题 → 空白 → 方法 |
| Pattern B | 应用重要性 → 技术挑战 → 现有方法 → 局限 → 新框架 |
| Pattern C | 科学观察 → 未解现象 → 研究问题 → 调查 |

### 阶段三：贡献提取引擎

将贡献分类为 8 种类型：

1. **新架构**（New Architecture）
2. **新损失函数**（New Loss Function）
3. **新训练策略**（New Training Strategy）
4. **新数据集**（New Dataset）
5. **新理论**（New Theory）
6. **新评估协议**（New Evaluation Protocol）
7. **新应用**（New Application）
8. **框架集成**（Framework Integration）

### 阶段四：语言风格画像

构建风格画像，包含量化指标：

- 句子长度分布
- 过渡词频率
- 动词偏好
- 修辞策略提取（新颖性策略、鲁棒性策略、泛化策略等）

---

## 方法章节逆向工程

提取目标论文的方法呈现逻辑：
- 设计动机 → 数学公式 → 实现细节 → 与其他模块的连接
- 提取训练策略与优化选择

---

## 实验策略挖掘

逆向分析实验设计：
- 基线选择逻辑
- 消融设计模式（单变量 vs. 累积 vs. 跨数据集）
- 结果呈现策略

---

## 结果分析逆向工程

提取：数值报告风格 → 比较逻辑 → 归因模式 → 解释深度。不复制数据，只复用分析框架。

---

## 图表标题风格提取

分析图表标题的句式模式、信息密度、命名惯例。

---

## 讨论风格建模

提取讨论章节的逻辑结构：为什么出现这些结果 → 与先前工作的一致性/差异 → 隐含意义 → 推广性声明。

---

## 期刊指纹识别

构建期刊特定风格向量：

| 期刊/会议 | 风格向量 |
|----------|---------|
| IEEE | 简洁、技术性、高密度、低叙事 |
| ACM | 算法导向、形式化、结构化 |
| Elsevier | 平衡、详细讨论、适度解释 |
| Medical Image Analysis | 临床动机、统计验证、可解释性讨论 |
| Nature | 高层意义、科学影响、广泛含义 |
| CVPR/ICCV/ECCV | 视觉证据、强消融、增量新颖框架、基准导向 |
| NeurIPS/ICML/ICLR | 理论动机、数学严谨、泛化讨论、证明导向 |

---

## 风格相似度控制目标

| 维度 | 目标相似度 |
|------|-----------|
| 逻辑相似度 | 80–95% |
| 结构相似度 | 70–90% |
| 语言相似度 | ≤30% |
| 句子相似度 | ≤15% |
| 直接短语复用 | 0% |

---

## 反抄袭约束

禁止复制：句子、段落、标题、贡献声明、摘要、章节过渡、公式解释。即使明确要求也不复制。提取模式 → 独立重构。

---

## 多论文融合模式

当提供多篇论文时：构建 A 的引言逻辑 + B 的方法逻辑 + C 的实验逻辑 + D 的讨论逻辑 → 融合生成新风格画像，避免任何单篇论文占主导。

---

## 审稿人仿真层

生成后评估：审稿人能识别出复制文本吗？新颖性看起来有理由吗？证据支持结论吗？能通过相似度检测吗？任何不确定 → 重写。

---

## 输出模式

| 模式 | 说明 |
|------|------|
| Mode 1 | 风格分析报告 |
| Mode 2 | 章节级仿写 |
| Mode 3 | 全文仿写 |
| Mode 4 | 期刊特定仿写（默认） |
| Mode 5 | 作者特定仿写 |

优先级：逻辑 > 结构 > 论证 > 语言。绝不将措辞模仿优先于科学推理模仿。
